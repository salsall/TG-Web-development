Opdracht 7

Maak nu een blok met width: 100px, height: 100px, margin: 5px, padding 5px en border-width: 5px. Zie hieronder:

Dit is de content
  1. Hoe breed is de box?
  2. Hoe hoog is de box?
  3. Wat merk je op met width en height?

Antwoord 7

De breedte is width + 2 * padding + 2 * border = 100 + 2*5 + 2*2 = 114px

De hoogte is height + 2 * padding + 2 * border = 100 + 2*5 + 2*2 = 114px

De width en height (breedte en hoogte) hebben dezelfde grootte, dus heeft de box de vorm van een vierkant. Ook is de totale width niet gelijk aan de width van 100 pixels.
