Opdracht 3

Er zijn verschillende selectors, bijvoorbeeld;

- type selectors

- descendant selectors
- class selectors

Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

Antwoord 3

Een type selector komt overeen met de naam van een HTML element waarbij al die elementen dan op dezelfde manier worden gestyled. Het voordeel hieraan is dat automatisch al die specifieke elementen op dezelfde manier worden gestyled. Bij dit voorbeeld wordt de tekst dat getypt is tussen de h4-tags in het blauw weergegeven (kijk naar "type selectors")

Een desecendant selector selecteert bepaalde elementen die afstammen van een opgegeven element en bepaalt de styling van die elementen. In dit voorbeeld wordt bij elke paragraph die genesteld is in een div de achtergrondkleur geel weergegeven. (Kijk naar dit paragraaf)
div p {background-color: yellow;}

Een class selector selecteert alle elementen die vallen binnen een bepaalde class en zorgt voor dezelfde styling van deze elementen met dezelfde class. In dit voorbeeld heeft de tekst dat binnen een element staat met de class="menu" toegediend valt het lettertype Arial. Ook als er andere elementen zijn die dezelfde class zijn toegediend hebben de lettertype Arial.
