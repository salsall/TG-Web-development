Opdracht 2

CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

Antwoord 2

De CSS syntax bestaat uit een Selector en een Declaration block. De declaration block bestaat weer uit meerdere declarations met ieder een propertie en een value. Met de selector selecteer je het HTML element dat je wil aanpassen. Een element selecteren kan op drie manieren, bijv. specifiek per element of per class. De declarations zijn gescheiden van elkaar door een semi-colon en een declaration block heeft een accolade aan het begin en aan het eind.
