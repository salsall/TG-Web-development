Opdracht 1

Je kunt op 3 verschillende manieren CSS in je HTML inladen.
Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

Antwoord 1

1. External CSS

Het is mogelijk om een file te openen met daarin alleen CSS wat de layout van de website bepaald. Hierbij moet er wel worden verwezen naar de CSS stylesheet in de HTML-document.

2. Internal CSS

Als tweede mogelijkheid kan de CSS ook in het HTML-document worden gezet. Hierbij gebruik je het element <style> </style> na het head en voor het body element. In het element style kan alle CSS code worden geplaatst.

3. Inline CSS

Om een style aan een specifiek element toe te voegen, kan de CSS code ook in de tag van dat element worden toegevoegd. Dan geldt de opmaak alleen voor dat specifieke individuele element.
