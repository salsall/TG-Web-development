Opdracht 6

  • String: Een waarde is een string zodra de value tussen dubbele aanhalingstekens is gezet. "Hello" is een String.
  • Number: Een waarde is een Number zodra de value een getal is, bijvoorbeeld 3.
  • BigInt: Een BigInt lijkt veel op een Number. De waarde van een BigInt is een getal dat eindigt met een n, bijvoorbeeld 55n.
  • Symbol: Er wordt een unieke en anonieme value geproduceerd.
