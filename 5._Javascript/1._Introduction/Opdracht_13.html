Wat merk je op?

  • let string = "3";
  • let number = 3;
  • let combinedText = string + number console.log(combinedText);

Bij het uitprinten van deze variabele staan de waarden van de string en number achter elkaar geplakt, in tegenstelling tot het geval bij numbers waarbij de waarden bij elkaar worden op geteld.
