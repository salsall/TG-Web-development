Cascading

Cascading betekent dat volgorde uitmaakt.

Als je in hetzelfde document twee keer CSS voor hetzelfde element declareert, gebruikt je pagina automatisch de laatste gegeven waarde. De eerdere waarde wordt dus genegeerd.

Inheritance

Inheritance slaat op de waarden die wel/niet worden meegegeven aan geneste elementen.

Tekstgrootte en kleur wordt bijvoorbeeld wel meegegeven. Als je deze bepaalde waarden geeft voor bijv. een section element, dan krijgen paragraph elementen binnen dat section element automatisch dezelfde waarden.
