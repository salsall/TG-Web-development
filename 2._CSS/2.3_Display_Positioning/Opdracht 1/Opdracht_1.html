Opdracht 1

Welke position properties zijn er? Leg voor elke position property uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

Antwoord 1

  • Static - dit is de default waarde voor HTML elementen, dus de properties top, right, bottom, left en z-index hebben geen invloed op het element bij deze property.
  • Relative - het element kan worden verplaatst ten opzichte van waar het normaal in het document zou staan door de properties top, right, bottom en left te gebruiken.
  • Absolute - het element wordt niet op de standaard plek aangewezen in het HTML-document en ergens in het document geplaatst gebruikmakend van de properties top, right, left en bottom. Er wordt hierbij ook geen rekening gehouden met ruimte tussen andere elementen.
  • Fixed - het element wordt niet op de standaard plek aangewezen in het HTML-document en wordt ergens in het document op een vaste plek geplaatst binnen de viewport gebruikmakend van de properties top, right, left en bottom. Er wordt hierbij ook geen rekening gehouden met ruimte tussen het element met andere elementen.
  • Sticky - het element staat gepositioneerd op de standaard aangewezen plek in het HTML-document en vervolgens relatief geplaatst ten op zichte van de meest dichtst bijzijnde scroll positie. Het element blijft zichtbaar op het scherm, ook tijdens het omlaag en omhoog scrollen op de webpagina. Gebruikmakend van de properties top, right, left en bottom kan de positie van het element worden vastgesteld.
