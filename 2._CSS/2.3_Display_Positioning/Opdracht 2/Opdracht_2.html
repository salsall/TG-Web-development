Opdracht 2

Hoe zorg je er voor dat het <h1> element altijd 50px van de top en 50px van de rechterkant van het scherm zichtbaar is. Maak het <h1> element rood door een RGB kleurcodering toe te passen.

Antwoord 2

Om een element altijd op een bepaalde afstand van boven, links, rechts of onder van het scherm te laten 'plakken' is het handg om gebruik te maken van de property value sticky. Als het element een vaste plek op de webpagina moet aanhouden dan is de property value fixed van toepassing.
